🧠 ✔ COM for (modo aprendizado)
# verifica se tem pelo menos 1 número
tem_numero = False
for c in senha:
    if c.isdigit():
        tem_numero = True

# verifica se tem maiúscula
tem_maiuscula = False
for c in senha:
    if c.isupper():
        tem_maiuscula = True

# verifica se tem minúscula
tem_minuscula = False
for c in senha:
    if c.islower():
        tem_minuscula = True

# verifica se tem caractere especial
tem_especial = False
for c in senha:
    if not c.isalnum():
        tem_especial = True
⚡ ✔ COM any() (modo mais profissional)
tem_numero = any(c.isdigit() for c in senha)
tem_maiuscula = any(c.isupper() for c in senha)
tem_minuscula = any(c.islower() for c in senha)
tem_especial = any(not c.isalnum() for c in senha)



🧠 Forma de pensar (bem prática)

Antes de criar função, pergunta:

"esse valor vem de fora ou vou pedir aqui dentro?"
🎯 Resumo pra fixar
input dentro da função → sem parâmetro
valor vindo de fora → com parâmetro
